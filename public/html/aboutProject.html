Description of frameworks and APIs used

The JavaScript framework Vue 3 was used for the frontend of this project. Node.js was used for the backend / querying the database. The express Node.js framework Express was used to develop our custom API. The public Nominatim API was used by the frontend to search for locations by name and the retrieval of coordinates. This was needed to render the map properly and update the table as users scroll around on the map.

Six Things That We Learned
  1. Vue is reactive. Updating ref.value from the script propogates the change into the template.
  2. JSON.stringify() can be used to send an object as the body for a PUT request.
  3. Adding forms in Vue is extremely easy, and adding the "required" property on an input will result in an error message box appearing on the input if a user tries to submit the form with that input empty.
  4. It's easier to build a front-end to tally with the backend if the backend is well built or coded.
  5. SQL joins can be very helpful in querying databases. They help make you write less code.
